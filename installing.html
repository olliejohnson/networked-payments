Installing

Server

You first need to place down a computer with a wireless or ender modem on the side then you can install the program.

Installing Server

  1. Run

    wget https://github.com/olliejohnson/networked-payments/main/server/installer.lua installer.lua

  2. Run installer

You can then start the server with the server command

Client

You first need to place down a computer with a wireless or ender modem on the side then you can install the program.

Installing Client

  1. Run

    wget https://github.com/olliejohnson/networked-payments/main/client/installer.lua installer.lua
